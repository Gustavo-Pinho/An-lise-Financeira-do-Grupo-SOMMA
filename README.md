❓ Dúvidas Frequentes
O notebook não está carregando os dados. O que fazer?
Verifique se o arquivo Excel foi carregado corretamente. Na célula de carregamento, há uma instrução de upload — selecione o arquivo quando a janela aparecer.

Preciso pagar para usar o Google Colab?
Não. O Colab é gratuito para uso básico. Basta ter uma conta Google.

Posso baixar os gráficos?
Sim! Clique com o botão direito sobre qualquer gráfico gerado e selecione "Salvar imagem como...". Ou, se quiser salvar todos automaticamente, posso incluir uma célula de exportação no notebook.

O notebook pode ser aberto sem executar o código?
Sim, você pode navegar pelas células e ler o código e os comentários sem executar nada. Porém, os gráficos e resultados numéricos só aparecem após a execução.
